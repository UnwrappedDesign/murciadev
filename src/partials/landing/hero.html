This repo is the scaffolding we use at Unwrapped to create new Tailwind based HTML templates.

This scaffolding uses gulp to create workflows to make esier the development. These are other of the dependencies and technologies we use:

  1. Tailwind CSS

    As utility CSS framework

  2. Handlebars

    As template engine to join our layouts, partials into pages.

  3. Imagemin

    to minify images and optimice them for web.
